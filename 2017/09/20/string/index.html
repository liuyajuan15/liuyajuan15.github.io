PHP字符串

2017-09-20 PHP PHP 阅读0次 评论

一个字符串 string 就是由一系列的字符组成，其中每个字符等同于一个字节。这意味着 PHP 只能支持 256 的字符集，因此不支持 Unicode

一个字符串可以用以下四种方式表达：

  • 单引号
  • 双引号
  • heredoc 语法结构
  • nowdoc 语法结构（自 PHP 5.3.0 起）
